# 📚 Student Notes Portal

A Django-based web application that allows students to upload, manage, search, and download study notes.

## 🚀 Features

* User Registration & Login
* Secure Authentication
* Upload PDF Notes
* Browse Notes
* Search Notes by Title
* Download Notes
* Edit Uploaded Notes
* Delete Notes
* Personal Dashboard
* User Profile Page
* SQLite Database Integration

## 🛠️ Tech Stack

### Frontend

### Backend

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Falak-Mohsin18/student-notes-portal.git
```

Navigate to the project:

```bash
cd student-notes-portal
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run migrations:

```bash
python manage.py migrate
```

Start the server:

```bash
python manage.py runserver
```

Open:

```text
http://127.0.0.1:8000/
```
